TMSM WooCommerce Booking Thalasso
=================

WooCommerce Booking Thalasso for Thermes Marins de Saint-Malo

Features
-----------

Requires Advanced Custom Fields Pro v5.3.7+

* Register Post Types:
    * Accommodation
    * Package
    * Discovery
    * Spa Treatment
* Register Taxonomies:
    * Accommodation Type
    * Package Type
    * Trip Type
    * Discovery Type
    * Spa Treatment Type
* ACF Fields for the 4 Custom Post Types
